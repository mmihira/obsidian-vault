# Core Game Loops

## Labour Value Theory

### Resource Gathering

- Gathering resources
- Loot variety

### Crystal Farming & Production

- Plant crystal shards and grow crystals over time
- Harvest magic essence or break crystals for raw gems
- Build production chains powered by crystal energy
- Balance crystal growth (power/resources) vs monster spawning (danger)

## Loop Overview

The core loops interlock:

1. **Gather** → [[Sanctuary/Economy/Items & Resources|Resources]] from the world
2. **Farm** → [[Sanctuary/Crystals/Crystal Growing|Grow crystals]] for gems and power
3. **Explore** → [[Sanctuary/Core Loops/Exploration & Expansion|Push into fog]] to find new crystal networks
4. **Defend** → [[Sanctuary/Combat/Combat & Defense|Fight monsters]] that scale with your crystal farms
5. **Build** → [[Sanctuary/Town/Town Progression|Grow your town]] with NPCs and buildings
6. **Export** → [[Sanctuary/Economy/Export Import System|Ship gems]] through the portal for upgrades

## Loop Timescales

### Moment-to-Moment (Minutes)

Spray crystals → Harvest mature crops → Manage hunger/mana → Tend farm

### Session Loop (Hours)

Plan network assault → Deploy towers → Push toward emitter → Harvest wild resources → Destroy emitter → Bank resources → Expand farm with new cores → Upgrade equipment

### Campaign Loop (Days/Weeks)

Clear network tier → Gain power → Reach new tier → Descend depths for rare materials → Advance town → Unlock new crystal types → Tackle harder content → Progress toward [[Sanctuary/Progression/Central Rift|Central Rift]]
