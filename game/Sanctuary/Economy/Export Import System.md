# Export / Import System

## Export Mechanism

- The export terminal controls the export mechanism
- The terminal has a **chest** which the player inputs items selected for exporting
- The player executes the exporting via a switch
- There is a charge time for exporting

## Import Mechanism

- [[Sanctuary/Progression/Food System|Food]] arrives via the exporter every morning — enough for one day at first
- Food is expensive to import (human organic material costs a lot of gems to stabilise)

## Export / Import Upgrade Tree

- The export/import machine has its own upgrade tree
- Increase export capacity
- Earn corporation benefits
