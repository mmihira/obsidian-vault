# Portal System

## Portal Dynamics

- Portals by default allow limited transfer of entities — only gems and other small objects
- Researchers have found ways to use gems to allow transport of other items from Earth to dimension side
- It is possible to strengthen the portal from the dimension side to make it easier to move items to/from the portal

## Portal Advancement

- The portal provides the player's information table with statistics
- The map provides information but its range needs to be upgraded with the portal
- Must be provided with enough power/items to create permanent upgrades
