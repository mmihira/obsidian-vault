# Map Variety

## Fog Transparency

- The fog won't be completely dark — it will be somewhat transparent so the player has some sense of direction
- In later levels the fog density can increase to make it more challenging to navigate

## Map Elements

Think about variety:
- Lakes
- Rivers
- Trees
- Wand production trees
- Chests / loot / other discoveries
- Mining nodes
- Resource collection nodes
