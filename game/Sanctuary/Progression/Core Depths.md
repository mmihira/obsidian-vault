# Core Depths (Dungeon System)

## Overview

Cleared fog networks unlock **dungeon access** beneath destroyed emitters. Each destroyed emitter reveals an entrance to core depths — multi-floor dungeons with increasing difficulty that provide long-horizon macro progression.

## Depth Structure

| Floor Range | Difficulty | Rewards |
| ----------- | ---------- | ------- |
| 1-10 | Easy | Common resources, tutorials |
| 11-25 | Medium | Rare shards, pattern blueprints |
| 26-50 | Hard | Power shards, unique cores |
| 51+ | Extreme | Legendary items, endgame content |

## Progression

- Depths have **multiple floors** with increasing difficulty
- Progression through floors provides:
  - Rare crafting materials
  - Unique crystal shards
  - Lore/story elements
  - Permanent upgrades
- Dungeon levels represent macro progress over long horizons
- Similar in design intent to Stardew Valley's mine — a map feature the player can continuously attack or devote resources to upgrade

## Design Intent

- Provides a **slow level progress system** built into the world
- Gives players a persistent long-term goal alongside the network clearing loop
- Higher floors gate access to materials needed for advanced [[Sanctuary/Crystals/Crystal Growing#Geometry & Arrangement System|crystal patterns]] and [[Sanctuary/Combat/Combat & Defense#Invasion / Assault System|invasion defense]]
