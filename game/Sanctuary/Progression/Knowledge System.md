# Knowledge System

## Knowledge Acquisition

Actions in the game give experience:
- Harvesting items
- Killing enemies
- Conversations
- Farming
- Killing clouds

## Level Progression

- Getting enough knowledge will advance level

## Level Uses

### Skill Unlock

- Use level points + (potentially items) to learn new skills
- Reduce knowledge gain when player has lots of unused skill points

### Base Stat Unlock

- Alternatively can boost mana pool + regeneration rate infinitely
- However with exponentially decreasing effectiveness
