# Buildings

## Unlockable Buildings

### General Merchant Shop

- Contains common items necessary for the player

### Portal Comms Building

- Starts as a simple open station
- Reveals to the player the nature of the world map expansion system
- Should be easily buildable as it introduces a major game mechanic/goal

### Antifog Technologies

- Sells antifog buildings for gold
- **Fog repellent building** — when placed, provides fog repellent for at least X hours in a radius
- **Cloud consumer** — building which can be placed and consumes clouds to provide crystal power

### Common Barracks

- Initial barracks which everyone will sleep in initially before building houses for everyone
